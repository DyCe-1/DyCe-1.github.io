Das grösste Lego-Set bis 2019

Das grösste Lego-Set bis zum Jahr 2019 ist die UCS Millennium Falcon 75192 vom Lego Star wars. Das Set umfasst schöne 7541 Legosteine, was man merkt. Wenn man das Set kauft wiegt der Karton im Shop mit allem darum und daran 15kg!!! Das Grosse Set wurde im Sommer 2017 veröffentlicht und ist immer noch im Lego-Store vorhanden.

Das teuerste Lego-Set bis 2019

Das teuerste Lego-Set bis zum Jahr 2019 ist das Ultimative Millennium Falcon Sammlermodell. Das Set wird mit rund 4.500$ bewertet und ist der Traum von jedem Lego-Sammler.

Das kleinste Lego-Set bis 2019

Das kleinste Lego-Set bis zum Jahr 2019 ist eine kleine Dampflocke, es befasst um die 10 Lego-Steine und kostete 0.95£ und ist dazu auch das Billigste Lego-Set aller Zeiten.

Das beliebteste Lego-Set 2019

Das beliebteste Lego-Set bis zum Jahr 2019 ist Hogwarts Castle vom beliebtem Lego Harry Potter. Es hat ungefähr 6020 Lego-Steine und 4 Minifiguren. Der einzige Nachteil ist das es kaum ins Regal passt.
